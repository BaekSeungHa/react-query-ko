## `QueriesObserver`

`QueriesObserver`는 여러 쿼리를 관찰하는 데 사용할 수 있습니다.

```tsx
const observer = new QueriesObserver(queryClient, [
  { queryKey: ["post", 1], queryFn: fetchPost },
  { queryKey: ["post", 2], queryFn: fetchPost },
]);

const unsubscribe = observer.subscribe((result) => {
  console.log(result);
  unsubscribe();
});
```

**Options**

`QueriesObserver`의 옵션은 [`useQueries`](../../framework/react/reference/useQueries)와 정확히 동일합니다.
