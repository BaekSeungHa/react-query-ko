## `QueryObserver`

`QueryObserver`는 쿼리를 관찰하고 쿼리 간 전환을 할 수 있게 도와줍니다.

```tsx
const observer = new QueryObserver(queryClient, { queryKey: ["posts"] });

const unsubscribe = observer.subscribe((result) => {
  console.log(result);
  unsubscribe();
});
```

**Options**

`QueryObserver`의 옵션은 [`useQuery`](../../framework/react/reference/useQuery)와 동일합니다.
