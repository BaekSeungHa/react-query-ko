## Community Projects

React Query를 기반으로 추가 기능이나 향상된 개발자 경험을 제공하는 다양한 커뮤니티 프로젝트가 있습니다. 아래 프로젝트들은 알파벳 순서로 나열되어 있습니다. 이 목록에 추가하고 싶은 프로젝트가 있다면 PR을 열어주세요!

> 이 프로젝트들은 전적으로 커뮤니티에서 유지 관리됩니다. 이들 프로젝트에 대한 질문이 있으면 프로젝트 유지 관리자에게 문의하세요.

### batshit

특정 데이터 유형에 대해 일정 시간 내에 중복 요청을 제거하고 배치하는 배치 관리자

링크: [batshit](https://github.com/yornaath/batshit)

### Blitz

Next.js를 위한 완전한 풀스택 툴킷

링크: [Blitz](https://blitzjs.com/)

### GraphQL Code Generator

GraphQL 스키마에서 React Query 훅을 생성

링크: [GraphQL Code Generator](https://the-guild.dev/graphql/codegen)

### Http-wizard

타입스크립트 마법으로 엔드 투 엔드 타입 안전 Fastify API

링크: [Http-wizard](https://http-wizard.com)

### Kubb

모든 API에 대해 SDK를 생성

링크: [Kubb](https://www.kubb.dev/)

### NgQuery

Angular를 위한 쿼리 어댑터

링크: [NgQuery](https://ngneat.github.io/query/)

### Normy

데이터 페칭 라이브러리를 위한 자동 정규화 및 데이터 업데이트

링크: [Normy](https://github.com/klis87/normy)

### OpenAPI codegen

OpenAPI 스키마를 기반으로 코드를 생성하는 도구

링크: [OpenAPI codegen](https://github.com/fabien0102/openapi-codegen)

### OpenAPI React Query codegen

OpenAPI 사양 파일을 기반으로 TanStack Query 훅을 생성

링크: [OpenAPI React Query codegen](https://github.com/7nohe/openapi-react-query-codegen)

### OpenAPI zod client

OpenAPI 사양에서 zodios 클라이언트를 생성

링크: [OpenAPI zod client](https://github.com/astahmer/openapi-zod-client)

### openapi-fetch

### Suspensive React Query

React Query에 Suspense 지원을 추가하여 데이터 페칭을 더 간단하고 선언적으로 만듦

링크: [Suspensive React Query](https://suspensive.org/docs/react-query/motivation)
